# Tic-Tac-Toe <img src="./public/tic-tac-toe_md.png" alt="Tic-Tac-Toe" style="width:35px;"/>

##### 1. Open the terminal and navigate to your project folder

```bash
cd <path>/Tic-Tac-Toe
```

##### 2. Use npm to install the dependencies

```bash
npm install & npm i
```

##### 3. Run the project

```bash
npm run dev
```

##### 4. Build the project

```bash
npm run build
```
